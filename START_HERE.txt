  OBTENIR LE TOKEN WIKI.JS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Ouvre Wiki.js → Administration ⚙️
2. Menu latéral → "API Access"
3. Bouton "New API Key"
4. Configure :
   • Name : "Portail Documentaire"
   • Permissions : ✅ pages:read, pages:write, pages:manage
   • Expiration : Never
5. Clique "Create"
6. COPIE le token immédiatement (ne sera plus affiché)
   Format : wk_1234567890abcdefghijk...

  $ npm run dev              # Lance en mode dev (hot reload)

BUILD :
  $ npm run build            # Crée le build production (/dist)

MIGRATION :
  $ npm run migrate -- \
      --url=XXX --token=YYY  # Migre tous les documents

  $ npm run migrate -- \
      --url=XXX --token=YYY \
      --category=technical   # Migre une catégorie

  $ npm run migrate:dry-run \
      -- --url=XXX --token=YYY
                             # Test sans créer de pages

  $ vercel                   # Déploie sur Vercel
  $ docker build -t portail . && docker run -p 8080:80 portail
