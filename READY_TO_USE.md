## 🎯 Réponse à tes Questions

### Question 1 : "Il te faut quoi pour utiliser une instance du wikijs ?"

**Réponse :** **JUSTE l'URL et le TOKEN !**

✅ URL : `https://ton-wiki.com`  
✅ Token : Ta clé API Wiki.js

C'est tout ! Le reste se configure dans l'interface.

### Question 2 : "Je fais comment concrètement ?"

**Réponse :** **Installe + Paramètre + Voilà !**

```
1. Installe l'app (serveur, Vercel, ou Docker)
2. Ouvre l'app dans ton navigateur
3. Paramètres → Configure Wiki.js (URL + Token)
4. Paramètres → Migre les données (1 clic)
5. ✅ TERMINÉ !
```

**Guide complet :** [HOW_TO_DEPLOY.md](./HOW_TO_DEPLOY.md)

## 🚀 Pour Démarrer Maintenant

### Option A : Test Local (30 secondes)

```bash
npm install
npm run dev
# Ouvre http://localhost:5173
# Explore les 16 documents de démo !
```

### Option B : Déploiement Production (5-15 minutes)

**Vercel (Le plus simple) :**
```bash
# Push sur GitHub
git init && git add . && git commit -m "v1.0.0"
git push

# Déploie sur Vercel
vercel.com → Import project → Deploy
# ✅ En ligne en 2 minutes !
```

**VPS/Serveur :**
```bash
# Sur ton serveur
ssh user@serveur.com
cd /var/www
git clone <ton-repo>
cd portail-docs
npm install && npm run build
# Configure nginx (voir DEPLOYMENT.md)
# ✅ En ligne en 10 minutes !
```
