╔═══════════════════════════════════════════════════════════════════════╗
║  🔑 COMMENT OBTENIR LE TOKEN WIKI.JS                                 ║
╚═══════════════════════════════════════════════════════════════════════╝

  1. Ouvre Wiki.js → Administration ⚙️
  2. Menu "API Access"
  3. Clique "New API Key"
  4. Configure:
     • Name: "Portail Documentaire"
     • Permissions: pages:read, pages:write, pages:manage
     • Expiration: Never
  5. Crée et COPIE la clé (ne sera plus affichée)
  
  Format: wk_1234567890abcdefghijklmnopqrstuvwxyz
